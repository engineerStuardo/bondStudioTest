## Get started (Local development)

1. First clone the repo

   ```bash
   git clone git@github.com:engineerStuardo/bondStudioTest.git
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
    npx expo start
   ```
